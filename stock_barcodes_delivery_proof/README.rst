This module allows capturing photos as proof of delivery from the
barcode scanner interface for outgoing deliveries.

**Key Features:**

- Capture delivery photos directly from the barcode scanner interface
- Support for mobile device cameras and desktop webcams
- Choose between picking-level or line-level photo capture
- Photo carousel with fullscreen viewing
- Metadata tracking (capture date, user, notes)
- Integration with standard Odoo picking workflow

Configuration
=============

To configure this module, you need to:

1. Go to **Inventory > Configuration > Settings**
2. Scroll to the **Delivery Proof of Delivery** section
3. Enable **"Enable Delivery Proof Capture"**
4. Select the **Capture Level**:

   - **Per Picking**: Capture photos for the entire delivery
     (recommended for simple workflows)
   - **Per Line**: Capture photos for each product line individually
     (for detailed tracking)

5. Click **Save**

The feature will now be available in the barcode scanner interface for
all outgoing pickings (deliveries to customers).

Usage
=====

**From the Barcode Scanner Interface:**

1. Open an outgoing picking in the barcode scanner
2. Process the delivery normally (scan products, etc.)
3. Click the **"Photo"** button in the action bar
4. The camera interface will open:

   - Allow camera access when prompted by your browser
   - For line-level capture, optionally select a specific product line
   - Click the camera button to take a photo
   - Review the photo and click **"Use Photo"** or **"Retake"**

5. The photo will be saved and displayed in the gallery
6. You can capture multiple photos if needed
7. Continue with the picking validation as normal

**From the Picking Form:**

1. Open a delivery order (outgoing picking)
2. Go to the **"Delivery Proof"** tab
3. Click **"Add Photo"** to manually upload photos
4. View all captured photos in the list/kanban view
5. Photos show capture date, user, and can have notes added

**Viewing Photos:**

- Photos appear in a carousel with navigation controls
- Click on a photo to view it in fullscreen
- Use arrow keys or buttons to navigate between photos
- Delete unwanted photos using the trash icon (if not readonly)
